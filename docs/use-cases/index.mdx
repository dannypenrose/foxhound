# Use Cases

Real-world scenarios showing how Foxhound solves document search and analysis problems.

Each use case includes a complete workflow from ingestion to final output, with example commands you can adapt to your own data.

## In This Section

- [Workplace Disputes](./workplace-disputes) - Build chronological evidence packs for HR or legal review
- [Due Diligence](./due-diligence) - Search thousands of documents for red flags before a deal closes
- [Subject Access Requests](./subject-access-requests) - Locate every document mentioning a specific person
- [Internal Investigations](./internal-investigations) - Cross-reference communications to establish who knew what and when
- [Contract Review](./contract-review) - Semantic search across document sets for clauses and obligations
- [Research Synthesis](./research-synthesis) - Merge notes, papers, and meeting minutes to surface key themes
